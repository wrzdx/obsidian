## Child and Sibling Combinators

Когда нужно выбрать элементы не просто по классу/типу, но по их положению в HTML-дереве.

- `>` — child combinator (потомок **прямой**, уровень вниз)
- `+` — adjacent sibling combinator (сразу следующий элемент-сиблинґ) 
- `~` — general sibling combinator (любой следующий сиблинґ)

## Псевдо-селекторы

### Разница: псевдоклассы vs псевдоэлементы

- Псевдокласс (pseudo-class) — селектор, начинающийся с одного `:` (например, `:hover`), который выбирает **существующие** элементы в HTML на основании их состояния или позиции.
- Псевдоэлемент (pseudo-element) — селектор с двумя `::` (например, `::before`), который позволяет стилизовать или вставлять части, которых **нет** явно в HTML-разметке. 

### Псевдоклассы

Полезные виды:

- `:focus` — элемент, который выбран пользователем (например, через Tab или мышь). 
- `:hover` — элемент, над которым находится курсор мыши. 
- `:active` — элемент, по которому в данный момент происходит клик (зажат). 
- `:link` и `:visited` — выбор ссылок, соответственно, ещё не посещённых / уже посещённых. 
- Структурные псевдоклассы:
    
    - `:root` — корневой элемент документа (чаще всего `<html>`). 
    - `:first-child`, `:last-child` — первый/последний ребёнок у родителя. 
    - `:empty` — элемент без дочерних элементов.
    - `:only-child` — элемент, который является **единственным** ребёнком у родителя. 
    - `:nth-child(...)` — гибкий способ выбирать элементы по их порядку/периоду. 

### Псевдоэлементы
Примеры и применение:
- `::marker` — позволяет стилизовать маркеры списка (`<li>`). 
- `::first-letter` и `::first-line` — стилизация первой буквы / первой строки текста в элементе. 
- `::selection` — стилизовать выделенный пользователем текст.
- `::before` и `::after` — добавить контент перед/после элемента, без изменения HTML


## Селекторы по атрибутам

Когда нужно выбирать элементы не по типу/классу/ID, а по наличию или значению атрибута.

- `[attribute]` — выбирает элементы, у которых есть указанный атрибут (независимо от значения). 
- `selector[attribute]` — комбинирует: выбирает элементы определённого селектора + наличие атрибута.
- `[attribute="value"]` — точное совпадение значения атрибута. 

### Частичные совпадения

- `[attribute^="value"]` — атрибут начинается с указанного значения.
- `[attribute$="value"]` — атрибут заканчивается на указанное значение. 
- `[attribute*="value"]` — атрибут содержит указанную строку **где-то** внутри.
