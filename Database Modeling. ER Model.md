- **Data** – is a collection of *raw facts*, figures, numbers, words, or observations that can be processed, analyzed, and interpreted to gain meaningful insights and make decisions.

>[!abstract] DIKW Pyramid
>
>- Wisdom \[knowledge + insight\]
>  understanding, integration, applied, reflected upon, actionable, accumulated, principles, patterns, decision-making process
>- Knowledge \[information + meaning\] 
>  idea, learning, notion, concept, synthesized, compared, thought-out, discussed


| Level                                | Description                                                                                                                 | Example |
| ------------------------------------ | --------------------------------------------------------------------------------------------------------------------------- | ------- |
| **Wisdom**<br>knowledge insight      | understanding, integration, applied, reflected upon, actionable, accumulated, principles, patterns, decision-making process |         |

| **Information**<br>data<br>context   | organized, structured, categorized, useful, condensed, calculated                                                           |         |
| **Data**                             | individual facts, figures, signals, measurements                                                                            |         |
