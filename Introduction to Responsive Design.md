# Что это такое?

**Responsive Design** - создание сайтов подстраивающихся на изменения размеров браузера с целью чтобы они работали на любом устройстве.

Существуют способы для способы для достижения этого – сделать сайт достаточно гибким (flexlible) или изменяющих макет для определенных размеров.

Смартфоны редко буду меньше 320px, поэтому это будет нижней границей. Вверхнюю границу установить тяжело, существуют супер широкие экраны, для таких случаев лучше использовать `max-width` и центрирование.
