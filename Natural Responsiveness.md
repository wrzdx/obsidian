## Вступление

Предпочтительно полагаться на *flexbox* и *grid* макеты для создания адаптивных сайтов, нежели на те которые полностью перестраивают макет.

## Не всему нужен CSS

Чистый *HTML* без *CSS* уже адаптивный. 

## мета тег `viewport`
Чтобы браузер пользователей самостоятельно не увеличивал/уменьшал наш сайт, нужно добавить следующий код
```css
<meta name="viewport" content="width=device-width, initial-scale=1">
```
