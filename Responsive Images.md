# Основы

Самая частая проблема – это сохранение пропорций картинки, решение – не объявлять обы `width` и `height`, в таком случае один из них будет со значением `auto`, и будем сохрянять изначальные пропорции картинки.

# background-size, background-position and object-fit

- `background-size` + `background-position` предоставляют довольно много контроля над отображением изображения. Работают они на элементах у которых есть *background image* (для `img` тегов не работает). 
  Значения для `background-size` – `cover`, `contain`, `auto`, одно значение для ширины, если два то второе для высоты.
  Значения для `background-position` – `top`, `right`, `bottom`,`left`, `center`, первое значение отступ слева, второе отступ сверху, если второе не указано то 50%. Если третье значение определено, то оно отнимается от первое (например bottom right 50px, будет отсчитано 50px снизу), четвертое значение отнимается от второго.

- `object-fit` – как `background-size`, но работает для `img` тегов.
  Значения могут быть `fill` (по умолчанию), `contain`, `cover`, `none`, `scale-down`.

# Адаптивное изменение изображения

- Проблема сохранения по разному обрезанных версий изображения в зависимости от макета называется **art direction problem**.

- Проблема загрузки одного изображения с разными разрешениями в зависимости от размера экрана устройства называется **resolution switching problem**.

## Resolution Switching: Different sizes

Стандартный `img` позволяет толко указывать браузеру на истноник изображения:
```css
<img src="elva-fairy-800w.jpg" alt="Elva dressed as a fairy" />
``` 

Однако мы можем использовать два атрибута – `srcset` и `sizes` – для предоставления дополнительных источников и подсказок для правильного выбора между ними. 
```css
<img
  srcset="elva-fairy-480w.jpg 480w, elva-fairy-800w.jpg 800w"
  sizes="(width <= 600px) 480px,
         800px"
  src="elva-fairy-800w.jpg"
  alt="Elva dressed as a fairy" />
```
 
- `srcset` – объявляет список истоников – *<источник> <размер в пикселях>w*
- `sizes` – объявляет медиа запросы – *<условие> <размер изорбажения><любая мера длины кроме процентов>*, в последнем запросе условие можно не писать для значения по умолчанию. Если размера изображения нет в слотах `srcset`, то выбирается первый размером больше указанного и масштабируется до нужного размера.
